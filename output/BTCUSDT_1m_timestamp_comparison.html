Observations

  • The VisionDataClient interprets timestamps as the end of each minute period, while the raw data marks timestamps at the beginning of the period.
  • This creates a "one-minute shift" in the data when comparing raw Vision API data with processed data.
  • The first candle of the requested period is typically missing in the processed output.
